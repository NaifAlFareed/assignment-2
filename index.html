Projects

Classroom Reservation System (JavaFX)

A desktop application built using JavaFX to simplify classroom scheduling. It gives admins tools like “Manage Rooms” and “Manage Reservations”, and normal users can search for available rooms, submit booking requests, and view their booking history. It reduces scheduling conflicts for meetings, classes, and events by adding approval flow.

  • Admin panel (room + reservation management)
  • User panel (search / request / history)
  • Goal: avoid double-booked rooms on campus

KFUPM Events Hub

A centralized event platform for KFUPM. Organizers create events, admins approve them, and students discover, register, and get reminders in one place instead of chasing WhatsApp and posters everywhere. Attendance can be tracked with QR codes, and admins get engagement analytics.

  • Approve + publish events in one dashboard
  • Students browse “Suggested” and “Recently Added”
  • QR code check-in and attendance analytics
